Adding Barcodes for Printing

The barcodes can be searched in the ERP item master and we can create our own barcodes.

You can type the item code or barcode in the text box labled as Barcode / Itemcode. And press enter to initiate the search.

As you enter whole thirteen charectors and press enter, if the item is not in the item masteror error connecting ERP message boxes will appear. The connectivity error is only for remaining you that there is some probem with the connectivity. On the other hand message box will appear that do you want to print the barcode as a custome one.

This fecility without any message box you can do selecting the custome mode of printing.

Now you can either select the item double clicking or selecting and pressing 'Enter" key from the keyboard. Then the current item will appear in the printing list.
